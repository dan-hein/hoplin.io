Welcome to hoplin.io RabbitMQ client's documentation!
=====================================================

A lightweight RabbitMQ client for Java (built on top official rabittmq java client)


Ease of Use
-----------
hoplin.io is meant to be as simple as possible as well as having small dependency on external libraries.
A permissive MIT license allows using the library both in open-source and commercial projects.
